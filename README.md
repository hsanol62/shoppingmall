# shoppingmall
this is shipping mall website by using JSP

## 목적
>쇼핑몰의 기본적인 기능을 만들어 웹을 형성해본다.

## 구현 기능
-메인페이지(상품진열)
-로그인/회원가입
-Q&A게시판
-장바구니담기
-주문하기
-마이페이지(주문내역 확인)
-관리자페이지(주문내역확인, 회원정보확인)

## 역할
-JDBC 연결
-tomcat 설정 및 연결
-ERD 다이어그램 작성
-오라클 데이터 추가
-데이터베이스 연동
-코드분석 ppt 작성
